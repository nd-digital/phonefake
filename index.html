📱 PhoneFake — Guide

Cet outil simule un appareil mobile (iPhone ou Android) dans ton navigateur, en portrait ou paysage. Il sert à tester l'apparence et le comportement de tes propres applications web — qu'elles soient classiques, responsive, PWA installables ou autres.

Aperçu : 4 formats de simulation — iPhone, Android, iPad, Tablette Android — interchangeables d'un clic.

✨ Fonctionnalités

  • 2 modèles d'appareil : iPhone (notch + coins arrondis) ou Android (punch-hole + coins plus carrés).
  • Rotation : bascule portrait ↔ paysage en un clic, l'écran et les décorations s'adaptent.
  • Multi-applis : place tes propres applis dans des sous-dossiers à côté de cette page (chacune avec son index.html) et elles apparaissent dans le menu Mode web.
  • Compatible : sites classiques, responsive design, PWA installables, applis avec service worker — tout ce qui se sert dans un navigateur.
  • Accueil et Applis : reviens à un launcher minimal ou retrouve la liste des applis ouvertes.
  • Multilingue : interface en 5 langues (🇫🇷 français, 🇬🇧 anglais, 🇪🇸 espagnol, 🇮🇹 italien, 🇩🇪 allemand), sélectionnable via les drapeaux en haut.
  • Mode desktop : le sélecteur Mode web en haut de la page permet aussi d'ouvrir une appli dans un nouvel onglet du navigateur, en pleine largeur (rendu desktop sans simulation).

💡 Utilisation

  • Ajoute tes applis dans des sous-dossiers du même répertoire que ce fichier ; elles apparaissent automatiquement dans le sélecteur.
  • Choisis iPhone ou Android pour adapter le rendu de ton appli aux contraintes de chaque plateforme.
  • Utilise Rotation pour vérifier l'affichage paysage de ton interface.

🖥️ Usage local uniquement

PhoneFake est conçu pour un environnement de développement local (Laragon, MAMP, XAMPP, serveur Node, etc.). Ce n'est pas un service en ligne — il sert à tester rapidement tes propres applications sur ta machine, avant déploiement.

⌨️ Raccourcis clavier

  • 1 iPhone · 2 Android · 3 iPad · 4 Tablette
  • R Rotation · C Comparer · H Aide · A Accessibilité · Échap Fermer
